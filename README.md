# Sheryians Coding School - Cohort 3.0: Assignment 3 (Design 1)
## SunMall Potato Chips Landing Page

### 📝 Overview
This project is part of **Assignment 3** for the **Sheryians Coding School Cohort 3.0**. It is a vibrant, modern landing page featuring SunMall Potato Chips. The project focuses on translating a high-fidelity reference design into a functional web layout using pure HTML and CSS, specifically targeting complex overlapping elements, breakout components, and precise flexbox layouts.

### 🚀 Technologies Used
- **HTML5**: Semantic markup and layout structure
- **CSS3**: Advanced styling, Flexbox, custom gradients, and absolute positioning
- **FontAwesome**: For UI icons (shopping cart, navigation arrows, star ratings)

### 💡 Key Concepts Demonstrated
Throughout the creation of this project, several core front-end concepts were applied and mastered:
- **Flexbox Architecture**: Structuring a flexible split-screen hero section and a horizontal product carousel layout.
- **Breakout Elements & Absolute Positioning**: Using absolute positioning and negative margins to allow elements (like the chip bags) to "break out" of their parent cards, creating a dynamic, layered 3D aesthetic.
- **Textured CSS Backgrounds**: Using CSS `repeating-linear-gradient` to generate subtle diagonal stripe textures without needing external image assets.
- **Complex UI Alignment**: Perfectly aligning nested card content, including inline star ratings, pricing, and dynamic buttons utilizing `justify-content: space-between`.

### 📂 File Structure
- `index.html` - The main DOM structure of the webpage.
- `style.css` - All custom styling, absolute positioning rules, and layout logic.
- `assets/` - Directory containing the brand logo, potato chip bags, and other graphic assets.

### 🏃‍♂️ How to Run
Simply download or clone the repository, and double-click the `index.html` file to open it in any modern web browser (Google Chrome, Firefox, Safari, Edge). No build steps, frameworks, or servers are required!
